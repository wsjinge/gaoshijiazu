## 二、文件格式说明

### JSON 文件结构
```json
{
  "Sheet1": {
    "headers": ["第17代：修字派", ...],
    "rows": [
      {
        "第17代：修字派": "高在天",
        "第18代：学字派": "高协章",
        ...
      }
    ],
    "total": 73
  }
}
```

### 优势
1. **编程友好**: JSON 可被所有编程语言直接读取
2. **易于解析**: 标准 JSON 格式，无需特殊解析库
3. **可扩展**: 可轻松转换为其他格式（CSV、XML、数据库等）
4. **可查询**: 支持 JSON 查询语言（如 jq）
